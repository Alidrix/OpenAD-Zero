# OpenAD Zero v0.1.0-rc1

## Summary

OpenAD Zero v0.1.0-rc1 is the first release candidate of the safe-by-default Active Directory lab operations platform.

It provides mission creation, internal scope validation, Nmap discovery, NetExec SMB safe enumeration, Nuclei safe web exposure scanning, BloodHound / SharpHound evidence upload, BloodHound Explorer V1, Evidence Manager, Reporting Engine, Lab Operations, Timeline, Progress Score, Docker Compose deployment and CI quality checks.

## Who is this for?

- Cybersecurity students.
- Internal lab operators.
- CTF and training environments.
- Authorized internal assessment workflows.
- Demonstrations of safe-by-design AD assessment orchestration.

## What is included?

- Backend API.
- Frontend cockpit.
- Worker queue.
- Evidence workflow.
- Reporting workflow.
- Lab operations workflow.
- Optional BloodHound CE profile.
- QA and CI checks.

## What is intentionally not included?

- No automatic exploitation.
- No credential dumping.
- No LSASS dump.
- No DCSync.
- No pass-the-hash.
- No persistence.
- No EDR bypass.
- No lateral movement automation.
- No arbitrary shell command from the frontend.

## Upgrade / install

```bash
cp .env.example .env
make up-build
make migrate
make smoke
```

## Validation

```bash
make backend-test
make frontend-build
make security-check
make release-check
```

## Known limitations

- BloodHound CE integration requires external configuration.
- PDF reporting is not included.
- Multi-user authentication is not included.
- This release candidate is intended for controlled and authorized environments.

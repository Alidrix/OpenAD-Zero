# OpenAD Zero v0.2.0 Backlog

This backlog lists candidate improvements after v0.1.0-rc1.

## Product improvements

- Improve dashboard summary cards.
- Add richer filters for findings.
- Add evidence-to-report selection.
- Add report template customization.
- Add report export improvements.
- Add demo seed improvements.

## Security and governance

- Add optional authentication.
- Add role model for local multi-user use.
- Improve secret scanning documentation.
- Add more security-check rules.
- Add stronger frontend env validation.

## BloodHound

- Improve BloodHound unavailable states.
- Add richer AD summary cards.
- Add better pathfinding evidence snapshots.
- Add clearer graph error handling.

## Jobs and operations

- Improve cancellation status visibility.
- Add job duration charts.
- Add retry policy settings.
- Add worker queue dashboard.

## UX and accessibility

- Improve keyboard navigation.
- Improve empty states.
- Improve table responsiveness.
- Improve dark mode consistency.
- Add manual accessibility checklist.

## Explicitly not planned

- No credential dumping.
- No LSASS dump.
- No DCSync.
- No pass-the-hash.
- No persistence.
- No EDR bypass.
- No automatic exploitation.

# QA Guide

## Local quick check

```bash
make up-build
make migrate
make seed-demo
make smoke
```

## Backend tests

```bash
make backend-test
```

## Frontend build

```bash
make frontend-build
```

## E2E tests

```bash
make e2e
```

## Debug Playwright

```bash
make e2e-ui
make e2e-report
```
